Great that you are back. You are member of the group modmaint on
PAUSE. This means that you can add modules via PAUSE webserver. Your
additions will live in the database immediately after you add them and
will be added to the module list itself when I touch it the next time.

When you log onto PAUSE your menu should now have a new item "Register
a module". If you install a recent CPAN.pm and have Netscape running
on your X-Server, then you can also try registering via the new module
CPAN::Admin, a pretty efficient helper module for Module Lsit
Maintainance.

Let me know if you have any questions or wishes for a better interface.
