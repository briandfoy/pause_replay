    This module enriches the builtin rand() function to return integer
    values or even return a fixed value.

    $ perl -Mrandom=fixed,6 -e 'print "cheat rolling die... ", rand'

    $ perl -Mrandom=integer -e 'print "rolling die... ", 1+rand 6'
    rolling die... 3
