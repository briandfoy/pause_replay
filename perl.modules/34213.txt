  rationale:

    Gtk2::VolumeButton provides a Gtk2 widget similar to that one used
    in the Gnome Panel and the Muine and Rhythmbox music players. It's
    intended to display the volume and let the user change it, but it
    also can be used for similar things by simply replacing the buttons
    images.

    I've choosen the Gtk2::Ex:: namespace because the Gtk2:: namespace
    should be reserved for 'native' Gtk2 widgets like Gtk2::Window,
    Gtk2::Button etc. Otherwhise there may be conflicts if a
    VolumeButton widget will be added to Gtk2..
