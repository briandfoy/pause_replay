<b>Логистика</b> - "обслуживающая" часть бизнеса, охватывающая все процессы с
момента закупки сырья для его производства до момента продажи готовой продукции

<font face=Arial><b>Основной результат работы логистики Компании - оптимизация
расходов.</b><br>
<b>Решаемые задачи для достижения этой цели:</b><br>
  ∙  Минимизация времени в транзите;<br>
  ∙  Оптимизация запасов;<br>
  ∙  Создание системы оценки показателей собственной работы;<br>
  ∙  Снижение логистических рисков<br>
<br>
<b>Расходный бюджет.</b> Затраты на функционирование предприятия формируют статьи
расходов, которые складываются в определенный расходный бюджет. Принцип построения
расходного бюджета - один. Контроль двойной - со стороны руководителя отдела
(структурной единицы) и со стороны финансового отдела. Принцип формирования расходного
бюджета такой, что он позволит "Соединить" обязанности с ответственностью; Наладить
обмен информацией между структурными единицами организации, так как отсутствие
такого обмена будет тотчас же отражаться в расходах; Мотивировать сотрудников,
так как расходный бюджет является одним их показателей, оцениваемых для получения
премии.<br>
<br>
<b>Связь бюджета с мотивацией.</b> Каждый сотрудник должен знать, как формируется
расходный бюджет. Каждый сотрудник должен быть объективно заинтересован в росте
прибыли компании. Работа отдела логистики строится на основной формуле: "Прибыль
= Доход - Расход". Основное влияние логистики (в отличие от других структурных
единиц) на Расходную часть бюджета Компании. Следовательно, логисты больше всего
заинтересованы в правильном структурировании расходов<br>
<br>
<b>Принципы мотивации.</b> Каким образом можно объективно заставить всех работать
так, чтобы минимизировать расходы? Создание максимально объективной системы мотивации.
Примеры мотивации для сотрудников различного уровня и различных структурных единиц.<br>
<br>
<b>Замкнутая система: Структура - Бюджет - Мотивация.</b> Структура компании
- это фактически перечень должностных обязанностей, соответствующих той или иной
должности. Обязанности подразумевают ответственность. Ответственность контролируется
расходным бюджетом. Степень влияния каждого сотрудника на расходы компании учтена
в мотивации. Мотивация каждого сотрудника учитывает его обязанности и такова,
что расходный бюджет уменьшается, благодаря оптимальной работе каждого сотрудника.<br>
<br>
<b>Пример моделирования системы мотивации для конкретного предприятия</b><br>
<br>
<b>Пример построения логистической структуры компании</b>, имеющей большое количество
сборных грузов. Основные принципы построения в условиях меняющейся и непредсказуемой
внешней среды. Использование тех же принципов для компании, имеющей абсолютно
другие аспекты деятельности.<br>
<b>Разложение логистического процесса на цепочку виртуальных складов</b> с соответствующими
показателями. Оценка логистической деятельности при помощи системы виртуальных
складов. Минимизация потерь на каждом этапе логистического процесса.<br>
<br>
<b>Оптимальный выбор партнеров по бизнесу</b> - минимум расходов. Более 40 критериев,
собранных в одну таблицу дают возможность объективно оценить компанию, которая
предлагает услуги. Цена - необходимое, но не достаточное условие<br>
<br>
<b>Транспорт</b><br>
  ∙  Международные перевозки<br>
  ∙  Оператор мультимодальной перевозки<br>
  ∙  Конвенция о перевозке грузов<br>

<b>Склад. Оптимизация работы склада. Система моделирования работы склада. Основные
параметры оценки эффективности работы склада.</b><br>
<b>I группа:</b> Показатели, характеризующие эффективность использования складских
площадей;<br>
<b>II группа:</b> Показатели, характеризующие эффективность складских технологических
процессов;<br>
<b>III группа:</b> Показатели, характеризующие уровень сохранности грузов;<br>
<b>IV группа:</b> Экономические показатели, характеризующие общую эффективность
работы склада<br>
<b>Примеры моделирования различных (по назначению и удаленности от поставщиков)
складов.</b><br>
Моделирование складов для собственного производства. Методика построения модели.
Необходимые данные для построения модели. АВС анализ грузопотоков для определения
исходных данных при построении Модели. Моделирование зон склада:Зона Приемки,
Зона Хранения, Зона Комплектации, Зона Отгрузки.<br>
<br>
<b>Моделирование оптимальных складских процессов для каждой зоны склада:</b> Описание
процессов для построения модели, Определение внутренних связей для управления
процессами, Разработка системы мотивации персонала для оптимальной работы склада.
Управление логистическим процессом при помощи планирования<br>
