    Simulation::Automate is a set of modules which allows the user to
    automate running a simulator with various parameters and
    postprocessing the simulator output (e.g. data analysis, plotting,
    reporting). It can work with any simulator (in fact on any program,
    not only simulators), but simulators that accept text input files
    and produce text output files are better supported.

    I found no modules on CPAN that do something similar.

    Simulation, under which all kinds of simulation-related modules,
