    FONT::FT2 is a functional Perl extension to allow the manipulation
    of characters and fonts. Is intended to simplify image generation
    based on text data.

    Imager::font::freetype2 is biased towards generation of images for
    web publication. FONT::FT2 is intended to be used by typesetting
    applications and drawing applications like Gimp::Perl and Sodipodi.
    Also, FONT::FT2 is planned to support Postscript output (it already
    supports text and xpm output) for generation of printed pages for
    labels, banners, notices, and silk screening. Included in the
    examples are scripts for generating Gnome Mahjongg tilesets.
