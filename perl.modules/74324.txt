  rationale:

    This module makes it easy to delegate tasks in Catalyst Models to
    Moose Roles.

    DBIC-based Schemas can be modified by creating Roles which are
    automatically applied to the Result classes a build-time. This makes
    it possible to customize (moosified) Schemas without the need to
    change the files created by the Catalyst helper-scripts.

    The name "Catalyst::TraitFor::Model::DBIC::Schema::ResultRoles" was
    choosen because it is a "Catalyst" "Trait for" "DBIC" "Schemas"
    which applies "Roles" to "Result"-classes

    The current beta-version can be checked out at
