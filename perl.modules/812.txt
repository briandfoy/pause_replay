module:        NamedParms
details:       bnpf  Easily implement CGI-style named parameters
format:        tar/gz
documentation: embedded PODs
installation:  Makefile.PL

This module allows any sub to easily implement named parameters in the style
of CGI->header or CGI->cookie. By supplying an array of parameter names, the
sub will receive the supplied parameters either in an array or a hash. The
subs will continue to support the 'traditional' argument formats.
