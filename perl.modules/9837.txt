Description - Class::Runtime
Similar to Class::Loader except a cleaner interface and a few more useful 
methods. Not specifically designed for object creation but for the
loading of class/module at runtime, testing if class/module is loaded, 
calling method/functions of class (while loading class/module if necessary) 
and unloading class/module at runtime. Possibly support in a distributed 
environment in the future.
