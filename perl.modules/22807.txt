    XBin stands for "eXtended BIN" -- an extention to the normal
    raw-image BIN (a raw memory copy of the textmode video memory)
    files.

    It's a console graphics format somewhat similar to ANSI graphics.

    XBin features:

    + allows for binary images up to 65536 columns wide, and 65536
    lines high. + can have an alternate set of palette colors either in
    blink or in non-blink mode. + can have different textmode fonts from
    1 to 32 scanlines high, consisting of either 256 or 512 different
    characters. + can be compressed (RLE)
