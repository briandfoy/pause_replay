Yes, I know, this is complicated region on PAUSE and I hope we can let
it phase out, because it's really overly complating things without any
merit: that mailing-list-IDs like LWWWP may be registered as
maintainers of modules is (imho) a thing of the past. For many years
nobody has requested to register a module on a mailing list.

Just for the sake of completeness: *Your* menu on PAUSE should show an
item "Mailing list menu: select mailing list/Action". If you'd click
there, you'd get access to all the modules registered in the name of
LWWWP. There is a table that lists you as "associated" with the LWWWP
mailing list: only people in this table ever see this menu item.
