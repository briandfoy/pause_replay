Here short overview taken from README for Deus:

Deus is Perl documentation framework build around XML based DeusXML
language which describes programs and libraries APIs.

Full implementation of this framework should allow:

1) conversion from DeusXML to POD formats of documentation

2) conversion from POD formated documentation to DeusXML

3) autogeneration of Perl sources from DeusXML files

4) autogeneration of Perl test scripts from DeusXML files

5) autogeneration of DuesXML files from Perl sources

6) synchronization of changes in DeusXML, POD and Perl files

7) one way conversion from DuesXML to other formats suitable for
   documentation (HTML, TXT, LaTeX)
