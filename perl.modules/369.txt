I'm the maintainer for Curses::Widgets, and I would like to suggest a new
module that would extend its functionality:  Curses::Forms.

Name           DSLI  Description                                  Info
-------------  ----  -------------------------------------------- -----
Curses::Forms  cdph  Form management for Curses::Widgets          CORLISS

This module will allow the creation of a form object that will contain a list
of widgets, options, and screen location.  It should return, when activated, a
list of returned widget values in the widget navigation order.  It should also
provide alerts upon placement of a widget that exceeds the form's boundaries.
