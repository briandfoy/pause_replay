    elements possible. And more reason, this dose not provide any

    so.

    Most of this module's uniqueness is possibility of strong
    mathematical operation by RGB elements unit. Some existing modules
    of similar this are providing some arithmetic way with hex numbers,
    but this module, by using decimal numbers, is able to manipulate a

    be simple and reduce programing cost to code about color calculating
