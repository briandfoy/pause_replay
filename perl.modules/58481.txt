  rationale:

    I chose the namespace DateTime::NaturalLanguage because that is
    what this module does - it converts any number of seconds into a
    "natural" language representation. For example, the value 22 becomes
    "22 seconds". A larger value, like 300, becomes "3 minutes, 20
    seconds".

    I believe that this module should be listed in the module list
    because it is useful to those creating applications and tools that
    might want to display a more "natural" time - instead of saying
    something like "2000 seconds remaining", they could say something
    like "33 minutes, 20 seconds remaining".
