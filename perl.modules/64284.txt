Hi,

I would like to request the Collections:: name space. The main intention of
modules under that is to provide easy to use /high performance iterators. So
developers don't have to bother between the use of C style for vs. foreach,
or how to make a high performance detect element in an array of elements,
etc. I think this can save development (It does in my projects).

List of modules are:

Collections::OrderedCollection
Collections::Set
Collections::Colection
Collections::SortedCollection
Collections::Bag

Each single collection has its own feautres, for example Set doesn't allow
duplicated elemnts, sorted collections provides an always sorted collection.
