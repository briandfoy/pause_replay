  - session support with flat files / 3 different DB engines (MySQL, =

SQL for all DBs)

* Cookie support: read/write/delete cookies
* Attached variables: Move your data page to page without sessions (via =
links/actions or via cookies)
* Easy mail functions: send mail in one line in html format, plus =
attachments.
* Upload/Download functions: Upload files automathicly and download =
wished files via cgi script (additional lib available with webtools: you =
can set rate of download and check final status of downloaded client)
* Events: you can handle follow events in your system (linux/unix):
OnFlush,OnError,OnExit,OnTerm,OnTimeOut
* Html/SQL templates: easy reading/replacing normal and SQL templates. =
(SQL/normal templates, SQL variables, template variables)
* DES III encryptions/decryptions transimted in MIME style (now you can =
secure send/receive data to/from browser without any traps)
