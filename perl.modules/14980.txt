    In The Eagle Book (Programming Apache Modules with Perl and C) the
    authors (Stein and MacEachern) describe a dynamic Navigation Bar.
    The module has been registered as Apache::NavBar, although it does
    not exist in CPAN.

    The main features of the new module, Apache::NavBarDD, are as
    follows:

    * It allows both single and two level navigation bars.

    * It can be used to endow existing HTML pages with a navigation
    bar, or it can be called from mod_perl modules.

    * Provides a full object oriented interface.

    I have already contacted Lincoln Stein who gave this consent to
    distribute the package. In fact, he commented that "I would
    appreciate it if you could call it Apache::NavBar and put it in
    CPAN"; I reckon this might be problematic, hence my naming the
    module Apache::NavBarDD.
