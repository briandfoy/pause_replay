Data::Validator version 0.5
======================

Validator is a module to validate and/or transform data before
using it, or (typically) inserting it into a database. It provides a simple 
way of creating and accessing functions which allow you to check the value 
of an item of data in many different ways, and, if you desire, to transform 
it prior to final storage in your database. Note that missing values are now 
automatically transformed to undef, which is what DBI requires.
