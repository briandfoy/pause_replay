SlideMap_1_2.pm: a module to assist in mapping microarray slide features
to their source microtitre plates and wells.  This module includes a
unique mapping algorithm that can be altered to accomodate any cDNA
array spotter.  It currently supports five different types of spotters:
Molecular Dynamics MDII and MDIII, Intelligent Automation Systems IAS
spotter, Lucidea Array Spotter, and Stanford-style array spotters.
