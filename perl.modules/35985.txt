    =head1 NAME

    Renamer - Perl extension for performing interactive filename

    =head1 SYNOPSIS

    use Renamer;

    InteractiveRename();

    Recognizes switches: -d debug mode -v verbose mode

    E.g. $ cat mytest.pl #!/usr/bin/perl

    use strict; use warnings; use Renamer;

    # Rename any, all or no files in a directory passed in as @ARGV.
    InteractiveRename();

    $ mytest.pl -dv ~/mydir

    =head1 DESCRIPTION

    Interactive file renamer allows filename changes from within a
    user's favorite editor. Most useful when complicated or one-time
    repetitive changes to a directory must be made but programmatic
    solutions are probably not worth the time to implement. Ignores
    subdirectories, if any.
