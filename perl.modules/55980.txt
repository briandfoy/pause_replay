  rationale:

    WWW::Metaweb is an interface to the Metaweb APIs over HTTP: -
    Authentication API to log in for registered reads and writes. - MQL
    read / write APIs to perform read and write queries via the Metaweb
    Query Language - Translation API for accessing images and body text
    as either raw source or blurbs / thumbnails.

    It provides Perl structure to JSON transformation for queries and
    results as well as error handling for HTTP, MQL and authentication
    errors. In the future this module is planned to provide a more
    abstracted interface to Metaweb as well.

    The best example of a Metaweb is Freebase
    (http://www.freebase.com). "Freebase.com is home to a global
    knowledge base: a structured, searchable, writeable and editable
    database built by a community of contributors, and open to
    everyone."[1]
