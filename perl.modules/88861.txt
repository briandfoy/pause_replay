    MaxQuant is quantitative proteomics software written by Jeurgen Cox
    of the Mann lab in the Max Planck Institute. It outputs data in text
    format. The primary experimental evidence is in a file called
    evidence.txt. MaxQuant does some processing on this data to
    summarize peptides, modification and so on, and further statistics
    can be performed using the sister application, Perseus.

    However, reproducibility in complex experimental designs is not
    currently achieved by this software, and is the aim of
    Statistics::Reproducibility.

    Bio::MaxQuant::Evidence::Statistics will provide the statistical
    interface specifically to MaxQuant Evidence files, for
    Statistics::Reproducibility.
