modules: Net::ProxyMod and Net::Divert

Net::ProxyMod creates a small TCP proxy (also transparent) and 
allows you to create two functions to which packets going from 
the client to the server and vice versa will be passed to. This 
is a framework that makes it easy to modify and debug the packet's
payload on their way for testing purposes.

Net::Divert is an interface to FreeBSD divert sockets. This also
allows you to specify a function incoming packets will be passed
to. All kinds of packets can then be altered at will. This is 
again useful for testing existing applications.
