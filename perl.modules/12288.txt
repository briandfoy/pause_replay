I have written a couple of modules built off existing code. The first is
Net::POP3S which allows for secure connections to POP3 servers (pop3s). This
is built on the existing Net::POP3 module with very few changes. The second
module is CGI::Session::Secure which is a CGI::Session driver allowing
secured (encrypted via Crypt::Blowfish) persistent sessions. Both modules
draw heavily from existing modules (Net::POP3S from Net::POP3 and
CGI::Session::Secure from CGI::SecureState and CGI::Session::File).
